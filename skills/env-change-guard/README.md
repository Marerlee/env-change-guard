# env-change-guard

一个更适合新手的 OpenClaw 技能。

## 这个技能做什么

这个技能只做两件事：

1. 记录环境变量或系统变量改了什么
2. 用电脑小白能听懂的话解释这次改动会有什么影响

它不追求复杂流程，也不强调太多规则，重点就是：

- 改了什么，要记下来
- 改完会影响什么，要讲清楚

## 适合什么场景

- 给 Python / Node / Java 加 PATH
- 设置 `JAVA_HOME`
- 配置 ADB / Android SDK
- 修改系统变量
- 用户明确表示自己不懂电脑配置，希望解释简单一点

## 默认记录位置

`logs/env-change-log.md`

## 文件结构

- `SKILL.md` —— 技能主说明
- `README.md` —— 给人看的介绍

## 说明

这是一个极简版技能。

如果你只是希望代理在改环境变量时：
- 留下变更记录
- 顺便给新手解释影响

那它就够用了。
