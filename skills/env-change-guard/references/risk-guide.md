# 环境变量变更风险说明

## 低风险

- 新增一个进程级变量
- 新增一个几乎不会冲突的普通变量

## 中风险

- 修改已有的用户变量
- 给用户级 PATH 增加一个目录
- 修改只影响当前用户的启动配置

## 高风险

- 修改系统级 PATH
- 删除变量
- 直接覆盖 PATH
- 修改多个工具共用的重要变量
- 在用户自己都不清楚的机器上做整机级变更

## 不同风险下的处理要求

### 低风险
- 简要说明
- 记录修改前后
- 提醒是否需要重新打开终端

### 中风险
- 清楚说明原因
- 记录完整修改前后
- 明确给出回滚方法

### 高风险
- 明确标注“高风险”
- 说明可能导致什么问题
- 保留完整旧值
- 尽量只做最小改动
- 修改后立即给出回滚和验证方法
